# 📚 만화 정리기

만화 스캔본 정리, 이미지 편집, 권 묶기까지 한 번에 해결하는 로컬 웹 도구입니다.

![스크린샷](screenshot.png)

---

## ✨ 주요 기능

- 📁 **폴더 스캔** — 작품/화 구조 자동 인식
- ✂ **양면 분리** — 펼침 스캔 이미지를 좌/우 2장으로 분리
- 🖊 **이미지 편집** — 크롭, 흰색/검정 마스킹 (단건 또는 화 전체 일괄 적용)
- 📦 **권 묶기** — 여러 화를 묶어 폴더 / PDF / CBZ / ZIP으로 저장
- ✏ **파일명 일괄 변경** — 접두사, 번호 자릿수, 시작 번호 설정
- 🔄 **원본 자동 백업 & 복원** — 편집 시 자동 백업, 언제든 복원 가능

---

## 🖥 실행 환경

- Python 3.8 이상
- Windows / macOS / Linux

---

## ⚙ 설치 및 실행

**1. 라이브러리 설치** (선택 — 없어도 기본 기능은 동작)

```bash
pip install Pillow       # 이미지 편집 기능
pip install img2pdf      # PDF 무손실 변환
```

**2. 실행**

```bash
python manga_organizer.py
```

실행하면 브라우저가 자동으로 열립니다. (`http://localhost:7777`)

## 📖 사용법

### 폴더 열기
상단 경로 입력창에 만화 폴더 경로를 입력하거나 📂 버튼으로 탐색기를 열어 선택합니다.

### 지원하는 폴더 구조
```
작품명/
  ├── 1권 - 전/
  │     ├── 001.jpg
  │     └── 002.jpg
  └── 1권 - 후/
        └── 001.jpg
```

### 이미지 편집
썸네일을 **더블클릭**하면 편집 모달이 열립니다.

| 기능 | 단축키 |
|------|--------|
| 크롭 | `C` |
| 흰색 마스킹 | `W` |
| 검정 마스킹 | `B` |
| 이 이미지에만 적용 | `Ctrl+S` |
| 화 전체에 적용 | `Ctrl+Shift+S` |
| 이전/다음 이미지 | `←` / `→` |
| 닫기 | `Esc` |

### 양면 분리
1. 편집 모달에서 **✂ 분리선** 버튼 클릭
2. 노란 분할선을 드래그로 위치 조정
3. 방향 선택: `좌→우 a|b` (한국/서양) 또는 `좌←우 b|a` (일본)
4. **✂ 분리 실행** 클릭

분리된 파일(`005a.jpg`, `005b.jpg`)을 열면 **↩ 분리 복원** 버튼으로 원본 복구 가능.

### 권 묶기
1. 좌측 사이드바에서 화를 다중 선택 (Shift+클릭 또는 드래그)
2. **→ 권에 추가** 버튼 클릭
3. 우측 패널에서 저장 형식 선택 후 저장

---

## 📁 원본 백업

편집/분리 실행 시 같은 폴더 안 `_원본백업` 폴더에 자동 저장됩니다.
- 백업은 최초 1회만 저장 (이미 있으면 덮어쓰지 않음)
- 복원은 편집 모달의 **↩ 복원** 버튼 사용
